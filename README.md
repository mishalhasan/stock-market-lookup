# Stock Lookup Project

## Overview

The **Stock Market Lookup** is a tool that allows users to search for and retrieve real-time stock information for publicly traded companies in the US stock market. It provides key data such as current stock price, daily changes, historical trends, and company details, all in one convenient interface.

This project focuses on:

- Fetching and handling data from APIs
- Handling asynchronous operations in JavaScript
- Building responsive, interactive user interfaces
- Error handling for network and API responses

---

## Features

- Search for stocks by **symbol or company name**
- View **current price, daily highs/lows, and percentage change**
- Display **historical trends** (optional: charts)
- Handles errors gracefully:
  - Network errors
  - Invalid stock symbols
  - API response errors

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **API:** [Alpha Vantage](https://www.alphavantage.co/)
- **Styling:** Tailwind CSS
- **Version Control:** Git & GitHub
